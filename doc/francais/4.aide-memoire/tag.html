pandocreon:carta-genius [1]
Cet élément doit être place au plus haut niveau du document XML.

info [0, 1] ((carta-genius))
Informations sur le document.
  • author : nom de l'auteur
  • title : titre du document (par défaut : "Cards generated using Pandocréon Carta-Genius (http://www.pandocreon.com)"
  • subject : sujet
  • keywords : mots-clés
  • copyright : informations sur le copyright (et la licence)
  • version : version du document
  • lang : langue
  • note : toute autre information

pdf [0,1] ((carta-genius))
  • version : version du standard PDF [1.3, 1.4, 1.5]
  • master-password : mot de passe PDF administrateur
  • user-password : mot de passe PDF utilisateur

unit [0,1] ((carta-genius))
  • base : unité de base [mm, cm, in, pt]

back [0,1] ((carta-genius))
  • side : côté de retournement [NONE, WIDTH, HEIGHT]
  • reverse : inverser l'ordre des pages [YES, NO]

fonts [0,1] ((carta-genius))
Déclarations de polices de caractères Postscript Type 1.
  • default : nom de la police par défaut

font [0...x] ((fonts))
  • id : nom Postscript de la police
  • outline : chemin vers le fichier d'outline
  • metrics : chemin vers le fichier de metrics

path [0...x] ((fonts))
Ajoute un chemin de recherche des fichiers de police.
  • dir : chemin vers le répertoire

images [0,1] ((carta-genius))
Déclarations d'images et de masques d'image.

mask [0...x] ((images))
  • id : nom donné au masque d'image
  • file : chemin vers l'image (doit être une image PNG en noir et blanc)

image [0...x] ((images))
  • id : nom donné à l'image
  • file : chemin vers l'image
  • width : largeur par défaut [valeur]
  • height : hauteur par défaut [valeur]
  • mask : chemin vers l'image de masque de transparente (doit être une image PNG en noir et blanc)
  • mask-id : identifiant d'un masque d'image préalablement définie, à utiliser comme masque de transparence

path [0...x] ((images))
Ajoute un chemin de recherche des fichiers d'image.
  • dir : chemin vers le répertoire

templates [0,1] ((carta-genius))
Déclarations de gabarits.

template [0...x] ((template))
  • id : nom du gabarit
  • template : nom du ou des gabarits à traiter avant celui-ci

includes [0,1] ((carta-genius))
Déclarations de fichiers à inclure.

include [0...x] ((includes))
  • file : chemin vers le fichier à inclure

path [0...x] ((includes))
Ajoute un chemin de recherche des fichiers à inclure.
  • dir : chemin vers le répertoire

variables [0,1] ((carta-genius))
Déclaration de variables qui sont interpr&aecute;tées au début du document.

deck [1...x] ((carta-genius))
Ensemble de cartes.

paper [1] ((deck))
  • type : format de papier standard (A0, LETTER, ...)
  • landscape : mode paysage, inversion de la largeur et la hauteur du papier [YES, NO]
  • width : largeur du papier (si non standard) [valeur]
  • height : hauteur du papier (si non standard) [valeur]
  • margin : taille de marge sur tout le tour du papier [valeur]
  • margin-width : taille de marge à gauche et à droite (si non uniforme) [valeur]
  • margin-height : taille de marge en haut et en bas (si non uniforme) [valeur]

hidden-ditch [0,1] ((deck))
  • odd : couleur du fond-perdu des pages recto [couleur]
  • even : couleur du fond-perdu des pages verso [couleur]

cardsize [1] ((deck))
  • type : format de carte (papier standard A0, LETTER, ...)
  • landscape : mode paysage, inversion de la largeur et la hauteur des cartes [YES, NO]
  • width : largeur des cartes (si non standard) [valeur]
  • height : hauteur des cartes (si non standard) [valeur]
  • space : espacement entre les cartes, verticalement et horizontalement [valeur]
  • space-width : espacement horizontal entre les cartes (si non uniforme) [valeur]
  • space-height : espacement vertical entre les cartes (si non uniforme) [valeur]

card [1...x] ((deck))
  • count : nombre de fois où la carte doit s'afficher
  • template : nom du ou des gabarits à traiter avant la carte
  • back : nom du ou des gabarits servant de dos à la carte

circle [0...x] ((card, template))
  • x : abscisse du centre du cercle [valeur]
  • y : ordonnée du centre du cercle [valeur]
  • radius : rayon du centre [valeur]
  • line-width : épaisseur de la ligne [valeur]
  • line-color : couleur de la ligne [couleur]
  • fill-color : couleur de fond [couleur]
  • dash1 : longueur des pointillés pleins (doit être utilisé avec 'dash2') [valeur]
  • dash2 : longueur des pointillés vides (doit être utilisé avec 'dash1') [valeur]
  • opacity : opacité (0.0 <= x <= 1.0) [scalaire]
  • blendmode : méthode de mélange des couleurs (Multiply, Overlay, ...) [blendmode]

box [0...x] ((card, template))

  • height : hauteur du rectangle [valeur]
  • radius : rayon des coins du rectangle [valeur]
  • line-width : épaisseur de la ligne [valeur]

  • fill-color : couleur de fond [couleur]
  • rotation : angle de rotation en degrés [scalaire]
  • dash1 : longueur des pointillés pleins (doit être utilisé avec 'dash2') [valeur]
  • dash2 : longueur des pointillés vides (doit être utilisé avec 'dash1') [valeur]
  • opacity : opacité (0.0 <= x <= 1.0) [scalaire]
  • blendmode : méthode de mélange des couleurs (Multiply, Overlay, ...) [blendmode]

line [0...x] ((card, template))

  • width : épaisseur de la ligne [valeur]
  • color : couleur de la ligne [couleur]
  • dash1 : longueur des pointillés pleins (doit être utilisé avec 'dash2') [valeur]
  • dash2 : longueur des pointillés vides (doit être utilisé avec 'dash1') [valeur]

  • cp-y1 : ordonnée du premier point de controle [valeur]
  • cp-x2 : abscisse du second point de controle [valeur]
  • cp-y2 : ordonnée du second point de controle [valeur]
  • width : épaisseur de la ligne [valeur]

  • dash2 : longueur des pointillés vides (doit être utilisé avec 'dash1') [valeur]

  • id : nom de l'image déja déclarée
  • file : chemin vers l'image
  • mask : chemin vers l'image de masque de transparence
  • mask-id : nom du masque de transparence déja déclaré
  • entire : déformer l'image pour remplir la zone [YES, NO]
  • x : abscisse du coin inférieur-gauche de l'image [valeur]
  • y : ordonnée du coin inférieur-gauche de l'image [valeur]
  • width : largeur de l'image [valeur]
  • height : hauteur de l'image [valeur]
