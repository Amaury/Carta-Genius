NAME

Carta Genius - Playing cards PDF generator.

SYNOPSIS

carta-genius -i input_file.xml -o output_file.pdf [-p paper_type] [-w paper_width] [-h paper_height] [-l landscape_mode] [-m margin_size] [-x margin_width] [-y margin_height] [-b flip_side] [-v pdf_version] [-r inverse_order] [-e author] [-t title] [-d include_path] [-f font_path] [-g image_path] [-L log_file] [-q]

DESCRIPTION

Carta Genius is a software that helps you when you want to generate PDF files with playing cards inside. All output characteristics are set in the XML input file, but can be overriden with command line options.

OPTIONS

-i input_file.xml
The name of the XML input file. The file must be written regarding the Carta Genius' syntax.

-o output_file.pdf
The name of the PDF file to create.

-p paper_type
The type of paper to use. Possible values are: 4a0, 2a0, a0, a1, a2, a3, a4, a5, a6, a7, a8, a9, a10, ra0, ra1, ra2, ra3, ra4, sra0, sra1, sra2, sra3, sra4, b0, b1, b2, b3, b4, b5, b6, b7, b8, b9, b10, c0, c1, c2, c3, c4, c5, c6, c7, c8, c9, c10, letter, legal, ledger, semi-letter, executive, tabloid, dl, com10, monarch, e-a3, e-a3/2, e-a4, e-a4/2, e-a5, e-a5/2, e-a6, e-a7, e-b4, e-b4/2, e-b5, e-b6, e-b7, e-b8, id1, id2, id3, business-card.

-w paper_width, -h paper_height
The paper size, in case of uncommon paper. These two parameters must be used together, when the -p option is not used. Examples: 450mm, 32cm, 12in

-l landscape_mode
Set the paper in landscape mode. In fact, when this option is set to ``YES'', the paper's width and height are inverted. Possible values are: yes, YES, no, NO

-m margin_size
Set the paper's margin size, with the same size on width and height. Examples: 10mm, 0.5in

-x margin_width, -y margin_height
Set precise margin sizes. Examples: 1.2cm, 8mm

-b flip_side
Set the flip side. Could be set to ``NO'' (the cards doesn't have any back), ``WIDTH'' (paper flip on its vertical side) or ``HEIGHT'' (paper flip on its horizontal side). Possible values are: width, WIDTH, height, HEIGHT, no, NO

-v pdf_version
Set the PDF version. Possible values are: 1.3, 1.4, 1.5

-r inverse_order
Inverse the pages order (last page first, first page last) if set to ``YES''. Possible values are: yes, YES, no, NO

-e author
Set the PDF's author.

-t title
Set the PDF's title.

-d include_path
Set one or many path that the program must use to find include files. If there are many directories, they must be set together, separated with semi-colons (';' character). All directories must be ended with a '/' character (or a '\' character on Windows). Carta Genius try first to find include files in his current working directory, and after it tries in each defined include directories (and stops at the first match).

-f font_path
Set one or many path that the program must use to find fonts. If there are many directories, they must be set together, separated with semi-colons (';' character). All directories must be ended with a '/' character (or a '\' character on Windows).

-g image_path
Set one or many path that the program must use to find images. If there are many directories, they must be set together, separated with semi-colons (';' character). All directories must be ended with a '/' character (or a '\' character on Windows).

-L log_file
Set the file where all logs must be written in addition to the error output.

-q
Quiet output. No output log will be written on error output.

RETURN VALUE

0 (zero) if succes, another value if an error occurs.

VERSION

This manual page documents the 4.0.0 version of Carta Genius.

SEE ALSO

Carta Genius is a part of the Pandocreon project, which aims to provide some games (boardgmaes, card games, web games, ...) under free licenses. More information on the website http://www.pandocreon.com.

Carta Genius uses the PDFLib functions library to create PDF files. More information on http://www.pdflib.com.

The ghostscript program could be used to do some conversion from PDF to many formats. It is useful to create some bitmap pictures from PDF pages. If your PDF file contains transparency, use version 8.14 and above of ghostscript. More information on http://www.ghostscript.com.
